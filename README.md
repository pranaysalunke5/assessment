# Node.js + GraphQL + PostgreSQL Assessment Submission

This project is a complete backend application built on **Node.js**, implementing a **GraphQL API** with **JWT authentication**, **file upload**, **server-side PDF generation**, and **email services**, fulfilling all requirements of the Backend Developer Assessment.
